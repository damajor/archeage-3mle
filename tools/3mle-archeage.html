3MLE to Archeage converter

This utilty takes code exported from 3MLE and fixes it for use in Archeage

Steps taken are:

  • Converts octaves to be correct unless nooctave is checked
  • Converts velocity (volume) from Mabiogi levels to AAU levels unless novolume is checked
  • Converts N notes created by 3MLE optimization to their correct note with octave offset
  • Fixes note length at the start of new tracks if needed
